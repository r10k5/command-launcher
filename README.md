# Command Launcher

Минималистичный лаунчер команд для быстрого запуска файлов и программ.

## Описание

Простое приложение для запуска часто используемых программ через короткие команды.

## Использование

1. Запустите приложение: -
2. Добавьте команду через кнопку "Добавить команду"
3. Введите короткое имя (например: `chrome`)
4. Укажите полный путь к файлу
5. Теперь просто введите команду и нажмите Enter!
